# AI IT Support Assistant

## Description
This is a simple AI-powered IT support assistant that detects issues and creates tickets.

## Features
- Chat interface
- Issue classification
- Ticket creation

## Tech Stack
- Python (Flask)
- HTML/CSS/JavaScript

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run backend:
cd backend
python app.py

3. Open frontend:
Open frontend/index.html in browser
